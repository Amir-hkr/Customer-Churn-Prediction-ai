# 📊 Customer Churn Prediction System

A Machine Learning project that predicts customer churn using classification models and a Streamlit web app.

🇬🇧 English Explanation
📊 Customer Churn Prediction System

This project is a Machine Learning system that predicts whether a customer will leave (churn) or stay with a telecom company.

🔍 What does it take as input?

The model takes customer information such as:

Gender
Senior Citizen status
Partner status
Dependents status
Tenure (months of usage)
Monthly Charges
Total Charges
🎯 What does it output?

The model provides:

📉 Probability of customer churn
✅ Final decision:
Customer will stay
Customer will leave
🧠 Project Goal

The main goal is to help companies:

Identify at-risk customers
Reduce customer loss
Improve business decisions and retention strategies
🚀 How it works
Collect user/customer data
Data preprocessing (cleaning + encoding)
Feature scaling
Machine Learning prediction
Display results using Streamlit UI
